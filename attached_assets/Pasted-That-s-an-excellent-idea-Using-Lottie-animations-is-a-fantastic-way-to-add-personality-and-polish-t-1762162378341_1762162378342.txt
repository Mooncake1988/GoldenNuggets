### TailwindCSS and Animations

Tailwind handles this in two main ways:

1.  **Transitions & Transforms:** For simple effects, Tailwind's utility classes are perfect. You're already using this with the `hover-elevate` class I see in the codebase. This is great for:

      * **Hover effects:** `hover:scale-105`, `hover:shadow-lg`
      * **Transitions:** `transition-all duration-300 ease-in-out`
      * **Fades:** `opacity-0` to `opacity-100` for modals or popups.

2.  **Keyframe Animations:** For more complex, repeating animations, Tailwind has built-in utilities like `animate-spin` (which you use for loaders) and `animate-pulse`. You can also define your own custom animations in your `tailwind.config.ts` file and use them as utilities.

Lottie animations are a bit differentâ€”they aren't *part* of Tailwind, but they work *with* it. You would use a Lottie player library (like `lottie-react`) to render the animation, and then use Tailwind's classes to control its size, position, and visibility (e.g., `<Lottie className="w-64 h-64" ... />`).

### How to Add Lottie to Your React App

The most popular library is `lottie-react`.

1.  **Install it:**

    ```bash
    npm install lottie-react
    ```

2.  **Get an Animation:** Find a free animation on [LottieFiles.com](https://lottiefiles.com) and download the `.json` file. Add it to your `client/src` folder.

3.  **Use it in a component:**

    ```jsx
    // For example, in your Categories.tsx loading state
    import Lottie from "lottie-react";
    import mapLoadingAnimation from "@/assets/animations/map-loading.json"; // (You would add this file)

    // ...
    if (isLoading) {
      return (
        <div className="min-h-screen flex items-center justify-center">
          {/* <Loader2 className="h-8 w-8 animate-spin" /> */}
          <Lottie animationData={mapLoadingAnimation} loop={true} className="w-48 h-48" />
        </div>
      );
    }
    // ...
    ```
